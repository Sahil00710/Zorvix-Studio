# API Routes

## Public

- `POST /api/contact`
- `GET /health`

## Admin Auth

- `POST /api/admin/login`
- `POST /api/admin/logout`
- `GET /api/admin/me`

## Admin Inquiries

- `GET /api/admin/inquiries`
- `GET /api/admin/inquiries/:id`
- `PATCH /api/admin/inquiries/:id/status`
- `DELETE /api/admin/inquiries/:id`
